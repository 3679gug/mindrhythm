# 🌞 마인드리듬 (MindRhythm)

**마인드리듬**은 일상적인 활동 패턴을 분석하여 사용자의 마음 건강(우울 위험도)을 체크하고, 맞춤형 리듬 미션을 통해 마음의 안정을 돕는 서비스입니다. 이 프로젝트는 논문구현의 일환으로 제작되었습니다.

## 🚀 주요 기능

### 1. 개인화된 마음 건강 분석 👤
- 나이, 성별 등 기본 정보와 함께 오늘의 무기력감, 활동 패턴, 생활 리듬의 규칙성을 설문합니다.
- 수집된 데이터는 머신러닝 모델을 통해 분석되어 **마음 리듬 분석 점수(100점 만점)**로 변환됩니다.

### 2. 정밀한 상태 진단 📊
- 분석 점수에 따라 4단계 상태(안정, 경도 의심, 중등도 의심, 강력 의심)를 진단합니다.
- 추정 MADRS(우울 증상 평정 척도) 지수를 산출하여 객관적인 지표를 제공합니다.
- 활동량, 규칙성, 무활동 비율 등 점수에 영향을 미친 주요 요인을 상세히 분석해 줍니다.

### 3. 오늘의 리듬 미션 ✨
- 사용자의 현재 점수에 맞춰 최적화된 미션(명상, 산책, 운동 등) 3가지를 제안합니다.
- 미션을 완료하면 '나의 마음 정원'에 예쁜 꽃이 피어나 시각적인 성취감을 느낄 수 있습니다.

## 🛠 기술 스택
- **Language**: Python
- **Framework**: Streamlit
- **Data Analysis & ML**: Pandas, NumPy, Scikit-learn (Joblib)
- **Visualization**: Plotly
